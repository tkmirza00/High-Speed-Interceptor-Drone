# High-Speed Interceptor Drone with Advanced Control for Precise Targeting

This repository contains the software and supporting material for a master's thesis project on a low-cost, high-speed autonomous interceptor UAV platform. The project combines a custom 3D-printed quadcopter frame, PX4 offboard control, ROS/ROS2-based autonomy, and onboard mmWave radar sensing for target detection and tracking.

## Project Overview

The aim of this project is to develop a lightweight, scalable, and easy-to-manufacture high-speed UAV platform capable of autonomous target approach and onboard radar-based tracking.

The platform uses:

- A custom 3D-printed rocket-shaped quadcopter body
- PX4-based flight control
- MAVROS offboard velocity control
- Raspberry Pi companion computer
- TI IWR6843ISK mmWave radar
- ROS/ROS2 radar processing and filtering
- Target-gated autonomous interception logic

The thesis demonstrates autonomous high-speed flight, onboard radar detection, and real-world balloon tracking tests.
